ROFL's Mod installer is designed to install modifactions into Minecraft.

Usage:

1. Click the plus (+) button
2. Select the zip file the mod is in
3. Repeat steps 1 and 2 until all mods are added to list
4. Click the merge button

!!
Note: Mods have to be installed in order, e.g. Modloader, then forge, then whatever you want
!!
